# cryptoChat

nodejs and angular2 fully end2end encrypted chat application

## get it up and running: 

### install redis on mac
* install homebrew (http://brew.sh/)
* brew install redis

### start backend
* (cli) redis-server
* go to ./server 
* npm install
* npm start

### start frontend
* go to ./client
* npm install
* npm start
